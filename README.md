# WheelVPN

yet another VPN client built on [Xray-core](https://github.com/XTLS/Xray-core). the client was created as an alternative to complex and "sponsored" clients like Happ and V2Ray.

## Build

```bash
./gradlew assembleRelease
```

req android ndk with `arm64-v8a` support.
